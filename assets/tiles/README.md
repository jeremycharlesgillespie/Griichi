# Tile Face Textures

Drop PNG files here to override the procedurally-generated tile faces.
Each file is named by the tile's `code_name` (case-sensitive):

| File         | Tile                         |
| ------------ | ---------------------------- |
| `Man1.png` … `Man9.png` | 1-9 of Characters  |
| `Pin1.png` … `Pin9.png` | 1-9 of Circles     |
| `Sou1.png` … `Sou9.png` | 1-9 of Bamboo      |
| `Ton.png`    | East Wind (東)               |
| `Nan.png`    | South Wind (南)              |
| `Shaa.png`   | West Wind (西)               |
| `Pei.png`    | North Wind (北)              |
| `Haku.png`   | White Dragon (白)            |
| `Hatsu.png`  | Green Dragon (發)            |
| `Chun.png`   | Red Dragon (中)              |

## Format

- Any aspect ratio is accepted, but **2:3 (e.g. 128×192, 256×384)** matches
  the in-game tile face and looks best.
- PNG with transparency is fine — transparent areas show the tile's body
  color (straw).
- File is loaded via Godot's `ResourceLoader`, so a `.png.import` sibling
  file will be auto-generated by the editor on first scan.

## Fallback

If a PNG is missing, the game generates one at startup using:

- Off-white background
- Border in the tile's suit color
- Unicode mahjong glyph (suit tiles) or CJK character (honors)

You can see all the auto-generated textures on the main menu preview.
